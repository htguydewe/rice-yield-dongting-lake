# Data

`processed/` 中保存的是论文最终 `run_111` 建模阶段使用的清洗后 CSV 数据，研究区域为洞庭湖生态经济区，适合随代码一起公开。

最终稿研究区是《洞庭湖生态经济区规划》范围内 33 个县（市、区），对应 2012-2021 年共 330 个县年样本和 2310 条月尺度记录。因此，表格中出现 33 个县是最终论文口径，不是另一个旧实验。为避免和本地历史目录名混淆，`run111_modeling_input/` 额外保存了第 111 轮的本轮输入表和训练/验证/测试划分。

主要文件：

- `月尺度数据_稳定耕地_清洗后.csv`：县域-年份-月份尺度的遥感、气象、地形等月尺度特征。
- `县年份建模样本_仅遥感气象地形.csv`：聚合到县域-年份尺度的基础建模样本。
- `县年份建模样本_清洗_农业机制变量.csv`：加入农业机制变量、历史滞后特征和目标编码后的建模样本。
- `变量覆盖报告.csv`、`机制变量匹配报告.csv`、`水稻单产匹配审计.csv`：数据质量与匹配过程审计文件。

`run111_modeling_input/` 中的主要文件：

- `run111_county_year_modeling_table.csv`：第 111 轮模型实际使用的县年机制变量表。
- `run111_train_samples.csv`、`run111_validation_samples.csv`、`run111_test_samples.csv`：第 111 轮训练集、验证集和测试集样本划分。
- `run111_manifest.json`、`run111_data_notes.md`：第 111 轮配置、指标和数据说明。

原始遥感影像、土地覆盖数据、DEM、行政区划、统计年鉴 PDF 等大文件没有放入本仓库。需要复现完整数据构建流程时，请先按论文说明准备原始数据，再运行 `code/data_preparation/` 中的脚本。
