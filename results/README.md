# Results

`final_figures/` contains the final thesis figures: Figure 1.1, Figure 2.1, Figure 3.1-3.3, and Figure 4.1-4.12. The study area is the Dongting Lake Ecological Economic Zone.

`model_outputs/` contains the final `run_111` model results:

- `run111/`: model metrics, train/validation/test split files, predictions, and residual details.
- Root-level metric/prediction CSV files mirror the run111 final results for convenient access.
- Feature-importance CSV files support interpretation of the thesis results.

Model binary files are not included. Use the scripts in `code/modeling/` if retraining is required.
